Navbar is not at the top of the page that make issues with user experience and web design, Navbar should always be the Top of Website of full width.

2. Logo is too big and not at correct position.

Logo must be added in the Navbar, that has small size but visible and placed at the top alongside navbar so that navbar and logo can be arranged correctly give a good loop to the header of the website.

3. Slideshow is not after the Header.

When we talk about Header it should only has Navbar Logo and Search bar only but slideshow should be placed after the Navbar so that website design can become more sophisticated and attractive. Planning a layout should be needed so that web design principals should follow.

4. Images are not captioned.

Images that are added in the page have no caption, no information about what they are and why they are, so caption should be provided to them. Captioning images creates a good impact in web design that shows the proper information of the data that we placed on website.
